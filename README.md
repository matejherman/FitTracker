# FitTracker

## Description
FitTracker is iOS application built using Swift and SwiftUI. Main purpose of the application is to track workouts and search for exercises with provided ExerciseDB API. The application manages to update users personal records on each exercise and tell the user his previous workout loads.

## Tech Stack
* **Language:** Swift
* **UI Framework:** SwiftUI
* **API:** RESTful API integration (ExerciseDB)

## Installation
clone this GitHub repository
```bash
git clone https://github.com/matejherman/FitTracker.git
```
After that open this app via **XCode** version at least version 26 and build

## Project status
I originalyl developed this project as a university semestral project and succesfully presented it. 
**Current status** The app's core logic and UI remain intact as a demonstration of my work with Swift and SwiftUI. However, due to structural changes made by the external provider to the ExerciseDB API, the live data fetching is currently non-functional.
